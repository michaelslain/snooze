# Snooze

### App for better sleep and focus

## Dev

**Stack**

-   Javascript
-   React JS
-   React Router
-   CSS
-   Sass
-   Electron

**Web**

```zsh
# enter web file
cd web-desktop

# install dependencies
yarn react-install

# run dev server
yarn react-start

# build for production
yarn react-build
```

**Desktop**

```zsh
# enter desktop file
cd web-desktop

# install dependencies
yarn

# run both react and electron server
yarn electron-start

# build electron app for production
yarn desktop-build

# build just react app for production
yarn react-build

# build electron app (without react) for production
yarn electron-build
```
